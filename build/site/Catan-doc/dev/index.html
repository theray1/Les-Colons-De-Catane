Catane ! C’est quoi ?

Pour rappel, Catane est un jeu de plateau dont voici un image :

Source : https://ladameenbois.fr/wp-content/uploads/2018/11/catan.jpg

Il peut se jouer jusqu’à 4 joueurs dans sa version de base. Etant donné que le jeu se compose de tuiles interchangeables, chaque partie est différentes de la précédente rendant ainsi l’expérience de jeu illimitée. Il existe plusieurs variantes au jeu, avec des cartes qui peuvent changer radicalement !
